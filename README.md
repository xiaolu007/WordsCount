WordsCount
===================================

实现功能
-----------------------------------
* 根据一个英文文档小文件生成大文件；   

* 查询大文件中中出现的不同单词；   

* 统计出这些单词出现的次数；   

* 按首字母A-Z顺序输出单词和对应出现次数。

调试环境
-----------------------------------
* JDK 1.8.0_77  

* Eclipse Mars.2 Release (4.5.2)   

* Maven 3.3.9  

mvn命令行运行
-----------------------------------
在windows的cmd下：
```
mvn exec:java -Dexec.mainClass=com.gh.WordsCount.WordsCount
